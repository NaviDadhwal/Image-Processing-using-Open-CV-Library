# Image-Processing-using-Open-CV-Library
Image Processing Using OpenCV Library

Overview

This project is a web-based image processing application built using Python, OpenCV, and Streamlit. It allows users to upload multiple images, apply various image processing operations, and download the processed results as a ZIP file.

The project was developed to explore practical image processing techniques and investigate how large numbers of images can be processed efficiently without relying on paid cloud services.

---

Features

- Upload one or multiple images
- Image Scaling / Resizing
- Grayscale Conversion
- Edge Detection
- Blur Effect
- Download processed images as a ZIP archive
- Simple and interactive Streamlit interface

---

Technologies Used

- Python
- OpenCV
- Streamlit
- NumPy

---

Project Workflow

1. User uploads one or more images.
2. Selected image processing operation is applied.
3. Images are processed using OpenCV.
4. Processed images are packaged into a ZIP file.
5. User downloads the results.

---

Supported Operations

Image Scaling

Resize images while maintaining usability and quality.

Grayscale Conversion

Convert colored images into grayscale format.

Edge Detection

Highlight image boundaries using OpenCV edge detection techniques.

Blur Effect

Apply blur filters to reduce noise and smooth image details.
